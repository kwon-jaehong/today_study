1) 목적 함수 (objective funtion) 
-> 함수 결과값을 최소, 최대화 할것이냐?

2) 변수 (decision variable or unkwon)
-> 변수 x를 목적함수에 넣고, 목적함수가 최소(최대)가 되는 변수 x를 찾는것임

3) 제약 조건 (constraints)
-> 제약 조건을 만족하는 x를 목적함수에 넣고 값을 봄


과정
-> 문제가 주어지면, 목적함수, 변수, 제약조건을 수학적으로 모델링하고, 다양한 최적화 알고리즘을 통해 해(변수 x)를 찾는다
